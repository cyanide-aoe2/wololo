# Collecting and Using Your Personal Data

## Types of Data Collected

### Personal Data

While using Our Service, the bot will use your Discord ID. Personally identifiable information may include, but is not limited to:

#### Discord ID:
The Discord ID is a randomly-generated number assigned to your Discord profile. This number is generated by Discord Inc. We will use this Discord ID to check if you have added yourself to our ignore lists. If added to our ignore list, the bot will discard all information received from your message.

#### Discord Server ID:
The Discord Server ID is a randomly-generated number assigned to your Discord Server. This number is generated by Discord Inc. We will use this Discord Server ID to check if you have specified muting/ignore behavior in your Discord server for the bot.

#### Other:
No other data is recorded, stored or forwarded.
